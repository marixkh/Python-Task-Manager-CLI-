# 📝 Python Task Manager (CLI)

A simple and practical daily task management application written in Python. This project was developed to practice File Handling and CRUD (Create, Read, Update, Delete) operations.

## ✨ Features
- **Add Tasks:** Permanently save tasks to a text file.
- **List Tasks:** Read and display all tasks with numbering.
- **Delete Tasks:** Remove completed tasks by their index number.
- **Search:** Quickly search through your tasks.
- **Data Persistence:** Uses `tasks.txt` to ensure your data is saved locally.

## 🚀 How to Run
To run the application, ensure you have Python installed, then run the following command in your terminal:
```bash
python task_manager.py
